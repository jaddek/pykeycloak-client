# pykeycloak-client

`pykeycloak-client` is an async Python client for Keycloak with token management, typed payloads, and service-layer APIs.

## What you get

- Async Keycloak API access
- Structured payloads, queries, and representations
- Service modules for auth, users, roles, sessions, clients, and authz
- MCP server adapter for tool-based integrations
